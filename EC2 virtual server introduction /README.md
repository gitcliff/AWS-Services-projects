#  7 - Introduction to EC2 Virtual Cloud Server


# 📖 Summary

Worked on **Docker installation and deployment on AWS EC2**, focusing on **container management and network configuration**.
Tasks included **installing Docker, managing user permissions, running a Redis container, pulling and running a custom app image, and configuring EC2 security groups** to solve **the challenge of deploying containerized applications accessible from the web**.
Set up **Docker and AWS Security Groups** to automate containerized app deployment and expose it securely, aiming to **enable scalable, portable, and accessible cloud applications**.

---

# 🛠️ Tools Used

* **Docker**: ✨ *Installed and configured on EC2 to run containerized applications.*
* **AWS EC2**: ⚙️ *Hosted Docker environment and managed security groups for network access.*
* **Docker Hub**: 🚀 *Source for pulling custom Docker images for deployment.*

---

# 🎯 Skills Gained

* **Docker Installation and Verification**: 💡 *Installed Docker, started daemon, and verified running processes.*
* **User Permission Management**: 🔍 *Added user to Docker group to run commands without sudo.*
* **Container Deployment and Port Mapping**: 🚀 *Ran Redis and a custom app container with exposed ports.*
* **AWS Security Group Configuration**: 🔐 *Modified inbound rules to allow public access on required ports.*

---

# ⚠️ Challenges Faced

* **Ensuring Docker daemon started properly**: 🛠️ *Used service commands and process verification.*
* **Linking Docker port mapping with AWS firewall rules**: 🔄 *Configured security groups to expose container ports externally.*

---

# 💡 Why It Matters

This lab demonstrates core DevOps skills for preparing cloud-hosted servers to run containerized workloads and expose them securely over the internet.
Mastering Docker installation, container deployment, and AWS networking is critical for building scalable, portable, and accessible cloud applications. 🚢☁️
